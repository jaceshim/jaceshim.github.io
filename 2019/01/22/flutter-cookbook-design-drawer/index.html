Flutter Cookbook Design / Add a Drawer to a screen 대해 정리

Flutter Cookbook Design / Add a Drawer to a screen - [원문](https://flutter.io/docs/cookbook/design/drawer)의 내용을 정리한다.

Scaffold

material library 에서 지원하는 기본 Layout 구조를 지원하는 구현체(클래스)
drawers, snack bars, bottom sheets을 지원한다.

Drawer

material library 에서 지원하는 수평 슬라이드 메뉴
Drawer를 사용하면 App상단에(AppBar) 햄버거 메뉴가 노출된다.

  • DrawerHeader : Drawer의 상단 영역
  • ListTile : 목록 형태의 Item
    • ListTile의 opTap() 이벤트를 감지하여 Navigator.pop(context) 를 통해서 Drawer의 display를 제어
