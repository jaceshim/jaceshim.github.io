Flutter Layout에 대해 정리

Flutter Layout - [원문](https://flutter.io/docs/development/ui/layout)의 내용을 정리한다.

Widget

  • 플루터의 Widget은 UI를 만드는데 사용되는 class 임.
  • Widget은 layout 및 UI요소를 만드는데 사용된다.
  • 플루터 layout 메커니즘의 핵심은 Widget 임, 거의 모든것이 Widget이고, 심지어 model 들도 Widget 임.
    • 보이는 Widget : images, icons, text
    • 보이지 않는 Widget : rows, columns, grids

Container

  • 컨테이너 위젯은 child 위젯을 커스터마이징 할 수 있는 위젯이다.
  • padding, margin, border, background color 등을 사용하여 이름을 지정하려면 컨테이너 위젯을 써야 한다.

Layout a Widget ( 위젯 배치 )

  • app 자체도 Widget 이다.
  • default banner, background color, drawers, snack bars, bottom sheets 등을 추가하기 위한 API가 존재하는 Material Components의 Scaffold를 사용하는게 가장 쉽다.
